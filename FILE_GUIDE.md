## ğŸ“ Model Directory (`model/`)

### Training Scripts

| File | Purpose | Size | Language |
|------|---------|------|----------|
| `train_model.py` | ML training script | ~8 KB | Python |

**What it does**:
1. Generates synthetic dataset (500 samples)
2. Performs EDA
3. Handles missing values
4. Scales features
5. Trains 4 ML models
6. Performs hyperparameter tuning
7. Evaluates models
8. Saves best model

**Output**: Generates `model.pkl`, `scaler.pkl`, `label_encoder.pkl`
